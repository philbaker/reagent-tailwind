# reframe-tailwind

A starting point with reframe and tailwind (plus purgecss to remove unused CSS).

## Startup

- Build/REPL: `npx shadow-cljs watch frontend` or `iced repl frontend`
- CSS: `npm run watch` or `npm run build` or `npm run build-prod`
